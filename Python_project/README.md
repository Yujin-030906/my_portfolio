# 🐶 Dog Image Viewer (강아지 사진 앨범 프로그램)

## 📌 프로젝트 개요

Python의 GUI 라이브러리를 활용하여
이미지를 탐색하고 사용자 인터랙션을 제공하는
간단한 사진 앨범 프로그램을 구현한 프로젝트입니다.

사용자는 버튼을 통해 이미지를 넘길 수 있으며,
특정 이미지를 선택하면 이벤트가 발생하는 구조로 설계되었습니다.

---

## 🛠️ 사용 기술

* Python
* tkinter (GUI 라이브러리)

## 🚀 실행 방법

1. 프로젝트 다운로드
2. `src` 폴더로 이동
3. 아래 명령어 실행

```bash
python project.py
```

---

## 🎮 주요 기능

### ▶ 이미지 탐색 기능

* "다음" / "이전" 버튼을 통해 이미지 순차 이동
* 이미지 인덱스를 활용한 순환 구조 구현

### 💙 사용자 이벤트 처리

* 하트 아이콘 클릭 시 메시지 출력
* tkinter 이벤트 바인딩 활용

### 📝 이미지 정보 출력

* 현재 이미지 파일명을 화면에 표시

---

## 💡 기술적 특징

* **tkinter 기반 GUI 설계**

  * Label, Button, Canvas 위젯 활용
* **이벤트 기반 프로그래밍**

  * 버튼 클릭 및 마우스 이벤트 처리
* **상태 관리**

  * 이미지 인덱스를 전역 변수로 관리하여 화면 업데이트
