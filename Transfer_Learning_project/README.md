# Transfer_Learning_project — 브라질 이커머스 분석 → 한국 쇼핑몰 적용

Olist 브라질 이커머스 데이터셋을 분석하고, 동일한 분석 구조를 한국 쇼핑몰 데이터에 적용한 대시보드 프로젝트입니다.

---

## 📂 파일 구성

| 파일 | 내용 |
|------|------|
| `olist.html` | 브라질 Olist 이커머스 대시보드 — 99,441건 주문 기반 매출·결제·카테고리 분석 |
| `olist_korea.html` | 한국 쇼핑몰 분석 대시보드 — 핵심 지표, 매출 & 주문 현황, 채널 & 카테고리 |
| `shopping_forecast.html` | ShopIQ 한국 쇼핑몰 종합 대시보드 — 매출·주문·고객·카테고리·예측 분석, 이상 탐지 |

---

## 📈 분석 내용

### 1. Olist E-Commerce Dashboard (`olist.html`)

브라질 이커머스 플랫폼 Olist의 공개 데이터셋 분석

- 핵심 지표 (Key Metrics) — 총 매출, 주문 수, 평균 주문액
- 월별 매출 추이 및 주문 건수 추이
- 결제 수단 분포 (Credit Card, Boleto, Voucher, Debit Card)
- 카테고리별 매출 비교

### 2. 한국 쇼핑몰 분석 대시보드 (`olist_korea.html`)

Olist 분석 구조를 한국 쇼핑몰 데이터에 적용

- 핵심 지표 (Key Metrics)
- 매출 & 주문 현황 — 월별 매출 추이, 주문 상태 분포
- 채널 & 카테고리 분석

### 3. ShopIQ 한국 쇼핑몰 종합 대시보드 (`shopping_forecast.html`)

사이드바 네비게이션 기반 멀티 페이지 대시보드

- **매출 분석** — 월별 매출 추이, 채널별 매출 비중
- **주문 분석** — 주문 상태 현황, 배송 완료·배송 중·주문 처리 분포
- **고객 분석** — 고객 세그먼트, 구매 패턴
- **카테고리** — 카테고리별 매출 및 성장률
- **예측 분석** — 매출 예측, 주문 예측, 이상 탐지 (결제 오류·배송 지연·반품률 상승 알림)
