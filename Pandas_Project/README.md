## 📊 프로젝트 상세

### Project.ipynb — 전기차 브랜드 가성비 종합 점수 분석

2025년 전기차 스펙 데이터를 기반으로 브랜드별 가성비 점수를 산출하고,
운전자 성향별 추천 모델을 시각화합니다.

**분석 내용**
- 주행거리 점수, 전비 점수, 배터리 점수를 합산한 종합 점수 산출
- 브랜드별 평균 가성비 비교 (Audi, BMW, Hyundai, Kia, Mercedes-Benz, Porsche, Tesla, Volkswagen, Volvo)
- 운전자 성향별(실속파 / 장거리파 / 패밀리·캠핑파 / 성격 급한파) 상위 10위 모델 시각화

**사용 기술:** Python, Pandas, Matplotlib, Seaborn

---

### mini_project1.ipynb — 운전자 성향별 전기차 추천

전기차 스펙 데이터를 분석하여 4가지 운전자 유형별 최적 모델을 시각화합니다.

| 유형 | 기준 |
|------|------|
| 실속파 | 전비 (주행거리 / 배터리 용량) 높을수록 |
| 장거리파 | 1회 충전 최대 주행거리 높을수록 |
| 패밀리·캠핑파 | 트렁크 용량 + 견인 능력 합산 높을수록 |
| 성격 급한파 | 급속 충전 출력 높을수록 + 제로백 낮을수록 |

**사용 기술:** Python, Pandas, Matplotlib, Seaborn

---

### Project2.ipynb — 지역별 주유소 휘발유 가격 시계열 분석

주유소 지역별 평균 보통 휘발유 판매가격 데이터를 활용한 시계열 분석입니다.

**분석 내용**
- 분기 데이터를 날짜(Timestamp)로 변환하여 시계열 인덱스 구성
- 서울, 부산, 대구, 인천, 광주, 대전 지역별 분기 유가 추이 비교
- 서울·부산 유가 변화 꺾은선 그래프 시각화

**사용 기술:** Python, Pandas, Matplotlib
