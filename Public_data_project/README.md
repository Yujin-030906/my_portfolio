# Public_data_project — 전국 기상 데이터 분석 및 시각화

기상청 공공데이터(강수량·기온·습도)를 활용해 기상 변화 추이와 폭우 발생 패턴을 분석하고 시각화한 프로젝트입니다.

---

## 📂 파일 구성

| 파일 | 주제 |
|------|------|
| `mini_project01.py` | 연도별 총 강수량 변화 추이 |
| `mini_project02.py` | 연도별 평균 기온과 폭우 발생 빈도 상관관계 |
| `mini_project03.py` | 연도별 1시간 최다강수량 분포 변화 (박스플롯) |
| `mini_project04.py` | 연도별 평균 습도와 폭우 발생 빈도 상관관계 |
| `mini_project05.py` | 기온·습도 vs 강수량 산점도 (3변수 복합 시각화) |

---

## 📊 데이터 소스

- `전국 강수량 2000 ~ 2025.csv` — 일별 전국 강수량 및 1시간 최다강수량
- `전국 기온 1997 ~ 2025.csv` — 일별 전국 평균 기온
- `전국 습도 2000 ~ 2025.csv` — 일별 전국 평균 습도

---

## 📈 분석 내용

### mini_project01.py — 연도별 총 강수량 변화

- 2000~2025년 연도별 총 강수량 합산
- 꺾은선 그래프로 연도별 강수량 추이 시각화

### mini_project02.py — 기온 vs 폭우 발생 빈도

- 폭우 기준: 1시간 최다강수량 80mm 이상
- 연평균 기온과 폭우 일수를 이중 축(twinx)으로 비교

### mini_project03.py — 1시간 최다강수량 분포 (박스플롯)

- 연도별 1시간 최다강수량 분포 변화를 박스플롯으로 시각화
- 이상값(극단적 폭우) 포함 표시

### mini_project04.py — 습도 vs 폭우 발생 빈도

- 연평균 습도와 폭우 일수를 이중 축(twinx)으로 비교
- 강수량·습도 데이터 병합 후 분석

### mini_project05.py — 기온·습도 vs 강수량 산점도

- 강수량·기온·습도 3개 데이터를 일별로 병합
- x축: 평균 기온 / y축: 1시간 최다강수량 / 색상: 평균 습도
- 3변수 복합 산점도로 기상 요소 간 관계 시각화
